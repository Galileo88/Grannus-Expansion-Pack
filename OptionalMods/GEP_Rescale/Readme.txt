## GEP_Rescale

GEP_Rescale is an optional config that rescales the GEP solar system to either 2.5x or 10x its normal size. Rescaling is accomplished without the use of Sigma Dimensions. GEP_Rescale works with any of the following installations:

  * JNSQ + GEP + GEP_Rescale
  * JNSQ + GPP + GPP_Secondary + GPP_Rescale + GEP + GEP_Rescale
  * GPP + GPP_Rescale + GEP + GEP_Rescale
  * GEP + GEP_Primary + GEP_Rescale
  * GEP + GEP_Primary + GEP_Rescale + GPP + GPP_Secondary + GPP_Rescale

For any other combination of planet packs, rescaling should be performed using Sigma Dimensions without GEP_Rescale.

To install GEP_Rescale:

1. Download and install Grannus Expansion Pack per the installation instructions.

2. Dowloand and install any other applicable planet packs and optional mods (see options above).

3. Drill down to [GEP Download]\Optional Mods\GEP_Rescale\ and select either Rescale_2.5X or Rescale_10X.

4. Copy the selected GameData folder into your KSP install.
